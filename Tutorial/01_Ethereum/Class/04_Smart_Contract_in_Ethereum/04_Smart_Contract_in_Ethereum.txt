*) Smart Contract:
    -> Smart contract is a program that run on ethereum blockchain network
    -> It's a collection of code (its functions) and data (its state) that resides at a specific address on the Ethereum blockchain.
    -> https://youtu.be/vZQaX2MsZT4?list=PLgPmWS2dQHW9FmqNqug3M5ooNuRqP-alu&t=120
    -> For Bitcoin coding language is Bitcoin Script
        -> Not Turing Complete
    -> For Ethereum coding language is Solidity
        -> Turing Complete
    -> Each node has the following:
        a) Current State of all Smart Contracts.
        b) History of both transaction and smart contract.
